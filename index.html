Dúvidas
Perguntas frequentes
Precisa instalar algum aplicativo? +
Não. O Painel Interativo funciona direto no navegador, para você e para os convidados. Basta escanear o QR code com o celular.
Posso usar em vários eventos por mês? +
Sim, sem limite de eventos. Você cria um painel para cada evento que realizar durante o período, quantas vezes quiser.
Como funciona o cancelamento? +
Você cancela a qualquer momento pelo MercadoPago. No plano mensal, o acesso continua até o fim do período já pago.
O que é a garantia de 30 dias do plano anual? +
Se nos primeiros 30 dias você não ficar satisfeito, devolvemos o valor integral sem burocracia.
Precisa de internet no evento? +
Sim, o painel funciona em tempo real. Recomendamos usar a rede do espaço ou um roteador dedicado para garantir estabilidade durante o evento.
